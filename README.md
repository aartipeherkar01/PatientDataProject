# PatientDataProject
Cleaned and preprocessed patient data by handling missing and inconsistent values. Merged two datasets into a single structured file. Applied group-by operations to extract insights and identify trends. Created graphs and visualizations to present key metrics and support data-driven decision-making.
In this patient data analytics project, I began by merging two datasets into a single CSV file and performing thorough data cleaning with Pandas. This involved identifying and handling missing values, creating new columns, and applying filters and slicers for refined data analysis. I then utilized group-by operations to extract meaningful insights and performed detailed data visualization using Seaborn and Matplotlib. These visualizations included various graph types, such as bar charts and scatter plots, to effectively present key metrics and support data-driven decision-making. Throughout the process, I followed essential data analytics steps, ensuring a comprehensive and insightful analysis.
